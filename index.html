Sayz

Say it. It's typed.

Push-to-talk dictation for macOS. Hold a key, talk, release. 100% on-device via the Apple Neural Engine. No cloud, no screen capture, no telemetry.

What it does

  • Push-to-talk and toggle modes with a configurable global hotkey.
  • Transcription to clipboard and the Sayz Scratchpad window.
  • Custom vocabulary substitutions, applied before the text is inserted.
  • Three Whisper model sizes (Tiny / Base / Small) selectable in Settings.
  • The direct-distribution build can additionally insert text at the cursor via the macOS Accessibility API; the Mac App Store build is scratchpad + clipboard only.

Privacy

Sayz never sends your voice anywhere.

  • Audio is processed on-device using Apple's Neural Engine via WhisperKit (CoreML).
  • No network calls during transcription.
  • No screen capture. Sayz never inspects window contents.
  • No telemetry, analytics, crash reporting, or usage data is collected.
  • Sayz only contacts the network for two explicit reasons: downloading a Whisper model, and (outside the Mac App Store) checking for app updates.
